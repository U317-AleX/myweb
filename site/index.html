1.

由 $$ f(p) = 0 $$ 知 $$ p^4 = -2p^2 + p + 3, 2p^2 = -p^4 + p + 3, $$ 则 $$ g_1(p) = (3 + p - 2p^2)^{1/4} = (p^4)^{1/4} = p, $$ $$ g_2(p) = ((p + 3 - p^4)/2)^{1/2} = (p^2)^{1/2} = p, $$ $$ Q.E.D. $$

2.

$$ g(x) = \pi + 0.5\sin(x/2), x \in [0, 2\pi], $$ 则 $$ g(x) \in [\pi, \pi + 0.5] \subset [0, 2\pi]. $$ 又 $$ |\frac{dg}{dx}| = |0.25\cos(x/2)| \leq 0.25 < 1, $$ $$ 故函数\,g(x)\,在\,[0, 2\pi]\,上有唯一的不动点. $$

fixed-point interation :
function [res, iteration] = fixed_point_iteration(f, TOL, p0, N0)
    arguments
        f {mustBefuntionHandle}
        TOL
        p0
        N0
    end

    p_pre = p0;
    i = 0;
    p = p0; 
    while i < N0
        p_pre = p;
        p = f(p);
        if abs(p-p_pre) < TOL
            res = p;
            iteration = i + 1;
            return
        end
        i = i + 1;
    end

    res = NaN;
    iteration = i;
    disp("the method was unsuccessful.");
end

function mustBefuntionHandle(f)
    if ~isa(f, "function_handle")
        error("function handle was request");
    end
end

function res = target_func3(x)
