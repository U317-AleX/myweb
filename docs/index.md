# 📚 我的学习笔记主页

欢迎来到我的学习笔记网站！这里记录了我在学习路上的思考与收获。

## 🧭 页面布局说明

- **左侧区域**  
  📂 主要是学习笔记分类与目录导航。  
  方便快速定位到不同主题、不同阶段的笔记。

- **右侧区域**  
  📝 详细展示当前笔记的内容。  
  包括学习总结、代码片段、思维导图、心得体会等。
